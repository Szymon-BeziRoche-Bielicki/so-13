# normalize
Prosty program command line do normalizacji wejściowej listy liczb do przedziału [0,1].

## Instalacja
Pobierz repozytorium oraz uruchom skrypt instalatora installer.sh. Podążaj za podanymi instrukcjami.

HTTPS:
https://github.com/Szymon-BeziRoche-Bielicki/so-13.git

GIT clone:
```
gh repo clone Szymon-BeziRoche-Bielicki/so-13
```
Następnie uruchom installer w terminalu:
```
chmod +x installer
installer
```

## Użytkowanie
```
normalize a b c ...
```

## Wymagania systemowe i zależności
Powłoka kompatybilna z POSIX

kompilator gcc
